## 规范性引用文件

ISO 6743-3:2003

工业润滑油分类标准，定义压缩机油（Class L-D），是所有润滑标准的基础框架

ISO 6521-1:2019 (preview)

规定往复式(活塞式)及滴油式旋转压缩机润滑油的分类与性能要求（DAA、DAB）

ISO/TS 6521-2:2021 (preview)

针对喷油螺杆和滑片压缩机润滑油，强调合成油（PAO、酯类等）的抗氧化、抗磨和清净性能。

ISO 8573-1:2010 (preview)

压缩空气质量分级标准，规定压缩空气中油含量、颗粒物和水分等级，对润滑系统设计具有约束作用。

ISO 10440-1:2007 (preview)

离心式（涡轮）压缩机标准，包含轴承润滑、密封油系统及润滑回路设计要求。

ISO 13631:2002 (preview)

石油天然气行业往复式压缩机标准，包含润滑系统配置、油品及运行控制要求。

ISO 8068:2006 

汽轮机润滑油规范，常用于离心压缩机轴承润滑油选型参考。

DIN 51506:2017 (德国)

压缩机润滑油分级标准（VCL、VDL等），按工作温度和抗氧化性能划分，广泛应用于欧洲工业领域。

ISO 4406:2021

润滑油清洁度分级标准，用于控制油液颗粒污染水平。

API 614:2021 (美国)

石化行业润滑、密封和控制油系统标准，适用于大型压缩机润滑系统设计。

API 617:2022 (美国)

离心式和轴流压缩机标准，其中包含轴承润滑、密封油系统和油品要求，通常与API 614配套使用。

API 618: 2014 (美国)

往复式压缩机标准，涵盖气缸润滑、注油系统、润滑油选择及控制要求。

Ingersoll Rand 润滑油运行条件规范

提供压缩机润滑油运行条件要求。

ISO 14224:2016 (preview)

石油天然气行业设备可靠性与维护数据采集标准，规定设备故障、维护及运行数据的分类与记录方法，可用于建立压缩机润滑相关故障数据库，为优化润滑策略和维护决策提供数据支持。

ISO 17359:2018 (preview)

设备状态监测与诊断通用指南，涵盖油液分析（Oil Condition Monitoring）在内的状态监测方法，为润滑状态评估、换油决策及预测性维护提供依据。
